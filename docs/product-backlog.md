# Product Backlog

## High Priority (Must Have)
- QR code attendance marking
- Secure login system with user roles
- Student attendance viewing
- Lecturer attendance monitoring
- Attendance data storage

## Medium Priority (Should Have)
- Attendance report generation (PDF)
- Admin dashboard for attendance management
- Upload absence justification
- Review and approve justifications
- Attendance analytics and charts

## Low Priority (Nice to Have)
- Notifications for low attendance
- Email/SMS alerts to students
- Reminders for attendance
- Parent notification system
- Appeal submission feature
