# User Stories

## Student
- As a student, I want to scan a QR code so that my attendance is recorded quickly and accurately.
- As a student, I want to view my attendance records so that I can track my progress.
- As a student, I want to receive notifications for low attendance so that I can take action early.
- As a student, I want to upload absence justifications so that valid absences are recorded.
- As a student, I want to receive confirmation after marking attendance so that I know it was successful.

## Lecturer
- As a lecturer, I want to generate a QR code so that students can mark attendance easily.
- As a lecturer, I want to view real-time attendance so that I can monitor participation.
- As a lecturer, I want to download attendance reports so that I can analyse student engagement.
- As a lecturer, I want to correct attendance records so that errors can be fixed.

## Administrative Staff
- As an admin, I want to manage attendance data so that records remain accurate.
- As an admin, I want to generate reports so that I can support academic decisions.
- As an admin, I want to send reminders to students so that attendance improves.

## Academic Coordinator
- As an academic coordinator, I want to review absence justifications so that approvals are managed properly.
- As an academic coordinator, I want to monitor attendance trends so that I can identify at-risk students.
